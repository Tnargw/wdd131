Overview

Purpose

The purpose of this site is to showcase my programming projects, skills, and background to potential employers and peers in a visually appealing and professional way.

Audience

The intended audience includes anyone interested in my programming work, but especially employers, peers, and anyone curious about the kind of projects I have built and the technologies I am familiar with.

Dynamic elements

JavaScript will be used to add interactive features including a light/dark mode toggle, and a search option that allows users to search for a project by name or by project tags.

Typography

Heading Font: Montserrat

Montserrat was chosen for headings because it is modern, clean, and professional. It stands out well at larger sizes, which makes it good for titles and section headers.

Paragraph Font: Open Sans

Open Sans was selected for the body text because it is very readable at smaller sizes and looks great on screens. It pairs well with Montserrat, and offers a clean and minimal style that doesn't distract from the main content.

Content

Home page

The home page will introduce visitors to my portfolio with a short welcome message and a preview of who I am. It will primarily focus on showcasing my programming projects. Each project will have a title, image, short description, and tags for the language used, project type, ETC. You will be able to filter projects by category using JavaScript. The page will also include a light/dark theme toggle button.

About Me

The about page will share more about me as a person and a developer. It will include some personal information, a photo, and some background on how I got into programming. I will describe my skills, education, and goals. This page will help visitors understand my personality, interests, and why I love programming.
